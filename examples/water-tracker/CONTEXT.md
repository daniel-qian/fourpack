# CONTEXT.md — 喝水记录小网页（名词表 + 事实）

> 免得你换个新对话又忘了这项目是干嘛的。

## 这是什么

一个**记录每天喝水量**的小网页。核心就三件事：加一杯水、看今天喝了多少、设一个每日目标。

## 名词表

- **一杯水**：默认 250 毫升（ml）。点一下"加一杯水"就给今天加 250ml；以后可以让用户自己改一杯多少。
- **今日总量**：今天到现在加起来喝了多少 ml。每天 0 点归零。
- **每日目标**：用户想每天喝够多少 ml（默认 2000ml）。到了目标页面要有反应。
- **记录**：每加一杯就存一条（时间 + 多少 ml），先存在浏览器本地。

## 已定方向

- 先做单页网页，纯前端，数据先**存在你这台浏览器自己里**（技术上这叫 localStorage——意思就是数据只存在你当前用的这个浏览器里，换台电脑或清掉浏览器缓存就没了，但做个自己用的小工具足够了）。不做后端、不做登录。
- 先把"加水 + 看总量 + 设目标"跑通，再考虑提醒、历史曲线。

## 还没定
